  • Protobuf 2.6(最新版本是3.4，下面会提到如何安装)

  • lxml（从网站 python第三方包下载网站）
  • tf Slim (tensorflow/models模块中包含了)
  • Jupyter notebook(如果不运行官网的网页测试就不需要)
  • Matplotlib
  • Tensorflow

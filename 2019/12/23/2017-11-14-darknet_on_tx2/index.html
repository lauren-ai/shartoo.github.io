首先知道yolo需要的几个数据

  • cfg
  • data
  • names
  • weights

其中前三个内容分别可以为:

1.1 obj.cfg

yolo的网络配置文件，一般可以从其官网 darknet-yolo-cfg文件下载一份，然后修改。不要自己手动创建，容易因为编码问题导致程序无法运行。

  • valid: 测试集图片路径。与train相同
